# Tostão

Seus tostões sob controle. Um aplicativo web simples e direto para controle financeiro pessoal.

## 🚀 Funcionalidades

* **Painel Principal:** Visão clara do salário base, despesas do mês e saldo restante.
* **Gestão de Despesas:** Adição e exclusão de gastos com atualização em tempo real.
* **Cofrinho:** Módulo para registro de economias, histórico de depósitos/retiradas e saldo total guardado.
* **Alertas Visuais:** O saldo muda de cor (verde, amarelo, vermelho) indicando a saúde financeira do mês.
* **Histórico Mensal:** Consulta de entradas e saídas de meses anteriores.
* **Armazenamento Local:** Os dados são salvos no navegador do usuário via `localStorage`.

## 🛠️ Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript (Vanilla)
